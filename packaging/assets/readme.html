simple-smtp-mailer - Read Me

Quick Start

After installation, simple-smtp-mailer will be available as a system service.

Configuration

Edit the configuration file:

/etc/simple-smtp-mailer/simple-smtp-mailer.conf

Service Management

Start the service:

sudo systemctl start simple-smtp-mailer

Enable at boot:

sudo systemctl enable simple-smtp-mailer
